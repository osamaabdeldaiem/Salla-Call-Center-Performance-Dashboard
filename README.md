# 📞 Salla Project – Call Center Performance Dashboard

An interactive **Power BI dashboard** developed to analyze call center performance, monitor service quality, evaluate agent productivity, and support data-driven operational decisions.

---

## 📌 Project Overview

This project provides a comprehensive analysis of call center operations using Power BI. It enables managers to monitor call volumes, service quality, agent performance, and operational efficiency through interactive dashboards and key performance indicators (KPIs).

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Pages

- 🏠 Home
- 📞 Call Center Overview
- 👨‍💼 Agent Performance
- ⏱️ Service Quality
- 📈 Call Trends

---

## 📈 Key Performance Indicators (KPIs)

- Total Calls Offered
- Total Calls Handled
- Calls Handled Within Threshold
- Abandoned Calls
- Service Level (%)
- Average Speed of Answer (ASA)
- Average Answer Time
- Call Answer Rate
- Agent Performance
- Forecasted Calls

---

## 💡 Business Insights

The dashboard enables users to:

- Monitor overall call center performance.
- Evaluate agent productivity and efficiency.
- Track service level and response time.
- Identify call abandonment trends.
- Compare forecasted calls with actual handled calls.
- Support operational planning and performance improvement.
